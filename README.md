BEM playground
==============

Online bemhtml compiler demo

### Install
 1. `git clone git://github.com/tadatuta/bem-playground.git`
 2. `cd bem-playground`
 3. `npm install && ./node_modules/.bin/bem make`

### Usage
#### Development environment
 1. `node app.js`
 2. `http://127.0.0.1:3000/`

#### Testing and production environment
 1. `make server`
 2. `http://127.0.0.1:3000/`
